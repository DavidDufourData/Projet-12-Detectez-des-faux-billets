# Détectez des faux billets

## Contexte et objectifs

L’objectif est de construire un modèle supervisé capable de distinguer les vrais billets des faux à partir de leurs caractéristiques géométriques.

## Soft Skills
- Approche scientifique et rigoureuse  
- Analyse critique des performances d’un modèle  
- Vulgarisation des résultats  
- Communication pédagogique  

## Compétences
- Régression logistique  
- Préparation du dataset (normalisation, train/test)  
- Clustering pour la pré-analyse  
- Évaluation via précision, rappel, matrice de confusion  

## Outils utilisés
- Python  
- Scikit-learn, Pandas, NumPy
